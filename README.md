# EDULedger System

## 📌 Description
A Django-based student record management system.

## ⚙️ Features
- Add student records
- Update and delete data
- Admin panel

## 🚀 How to Run
1. Clone the repository
2. Install dependencies
3. Run the server:
   python manage.py runserver
